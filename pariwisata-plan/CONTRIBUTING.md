# Contributing to [Nama Project]

Terima kasih atas minat Anda untuk berkontribusi! Dokumen ini berisi panduan untuk membantu Anda memulai.

## Tata Cara Kontribusi

### 1. Pelaporan Bug (Issues)
Jika Anda menemukan bug, silakan buat Issue baru dengan format:
- **Judul**: Deskripsi singkat bug
- **Langkah Reproduksi**: Cara untuk memunculkan bug tersebut
- **Ekspektasi**: Apa yang seharusnya terjadi
- **Screenshot**: (Opsional) Bukti visual

### 2. Pengajuan Fitur
Kami terbuka untuk ide baru! Buat issue dengan label `enhancement` dan jelaskan user value dari fitur tersebut.

### 3. Pull Requests (PR)
1.  **Fork** repository ini
2.  Buat **Branch** baru untuk fitur Anda (`git checkout -b fitur-baru`)
3.  **Commit** perubahan Anda (`git commit -m 'Menambahkan fitur baru'`)
4.  **Push** ke branch (`git push origin fitur-baru`)
5.  Buat **Pull Request** ke branch `main` repository ini

## Coding Standards

- **TypeScript**: Gunakan tipe data yang eksplisit (hindari `any`)
- **Tailwind**: Gunakan class utility standar. Hindari style inline
- **Komponen**: Pecah komponen besar menjadi komponen kecil dan reusable
- **Naming**: PascalCase untuk Component, camelCase untuk fungsi/variabel
